**A todo project made with create-react-app.**

Components used :

 - Material UI

Used Context API.

To start project ;

    cd client

    npm start
	    or
    yarn start

To start server ;

    cd server
    
    json-server --watch db.json --port 3004

(If you dont use port 3004 requests will return 404. You can change port in 

> Services/tasks-service.js
